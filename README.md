# GradeTrack
This project is a web-based CGPA calculator developed using Python and Flask. It enables students to register,log in, input course units and grades, and automatically compute their CGPA using a 5-point grading system. The application also stores user data securely using hashed passwords and maintains a history of CGPA calculations for each user.

**Features:**

1. User registration and login system

2. Secure password hashing using Flask-Bcrypt

3. CGPA calculation using 5-point grading system
4. Dynamic course input (add multiple courses)
5. CGPA history tracking per user
6. Responsive and clean UI using Bootstrap

**🛠️ Technologies Used:**

1. Python

2. Flask

3. SQLite (Database)

4. Flask-SQLAlchemy

5. Flask-Bcrypt

6. HTML, CSS, JavaScript

7. Bootstrap
